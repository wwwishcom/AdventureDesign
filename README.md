# AdventureDesign
# Bus-ing : 함께 만드는 발걸음, '도전'을 '일상'으로
교통약자의 이동 전 과정을 지원하는 시민참여 기반 실시간 장애물 지도 & Door-to-Door 내비게이션 서비스
## 🚀 Introduction

Bus-ing은 휠체어 사용자, 전동 스쿠터 이용자, 노약자, 유모차 사용자 등
이동에 제약이 있는 시민을 위한 시민참여형 공공 이동 지원 서비스입니다.

기존 지도 앱은 엘리베이터·경사로 같은 정적 정보만 제공하지만,
Bus-ing은 시민 제보 기반 실시간 동적 장애물(고장, 공사, 불법주차 등)을 반영하여
예측 불가능한 이동 위험을 줄이고 안정적인 이동을 가능하게 합니다.

특히 Bus-ing은 Door-to-Door 접근 방식을 통해
“집 문 앞에서 출발해 목적지 문 앞에 도착하기까지”
이동 전 과정을 하나의 여정으로 보고 지원합니다.

## Problems
### 1) 장애인 외출 과정에서의 예측 불가능성

기존 지도 앱은 정적 정보만 제공 → 실제 이동 중 마주치는 장애물 반영 불가

시민이 도로 위 장애물 정보를 간편하게 제보할 수 있는 시스템 부족

### 2) Door-to-Door 이동 안전 확보 불가

현행 저상버스 앱은 탑승 지원 기능에만 집중

실제 이동 전체 과정을 지원하는 서비스 부재

### 3) 심리적/물리적 장벽

탑승 의사 전달이 운전자에게 제대로 전달되지 않아 차별 발생

이동 과정에서의 불안감·낯섦으로 외출 자체를 포기하게 됨

## 🚪 Door-to-Door란?
아래는 교통약자 시점에서 실제 Door-to-Door가 필요한 이유를 보여줍니다.

### 1. 집 안에서 문밖으로 나오는 과정
- 복도 장애물, 경사 여부, 문턱 등 생활 공간의 물리적 제약

### 2. 엘리베이터 이용 여부
- 엘리베이터 고장 여부, 점검 중 여부는 실제 이동 성공 여부에 결정적

### 3. 아파트/집 주변 보행로 이동
- 불법 주정차, 턱, 공사장 등 실시간으로 변하는 동적 장애물

### 4. 정류장까지의 이동
- 경사도, 보도 상태, 우회 필요 여부

### 5. 저상버스 탑승 과정
- 운전자와의 의사소통 장벽
- 램프 작동 여부
- 인파, 탑승 대기 상황

### 6. 목적지 하차 후 건물 입구까지 이동
- 보행로 환경, 진입 경로의 안전성

### 7. 건물 내부 이동
- 내부 엘리베이터, 경사로, 위치 안내 등

Door-to-Door는 이 A→B 전체 여정을 하나의 “경험 단위”로 보며, 모든 구간을 연결해 지원하는 개념입니다.

## Solutions
### ✔ 시민참여형 실시간 장애물 지도

사용자가 도로 위 장애물을 사진/위치 기반으로 제보 → 실시간 장애물 지도 구축
데이터 신뢰도 평가 + 리워드 제공으로 참여 유도

### ✔ 장애물 우회 스마트 경로

제보 데이터를 활용해 **장애물 우회 경로**를 제공
예상 이동 시간 + 대체 이동 루트 안내

### ✔ 시민 자원봉사자 매칭

이동 보조가 필요한 이용자와
가까운 자원봉사자를 메시지/전화 등을 통해 실시간 연결

### ✔ 사회적 편견 완화

시민이 직접 제보에 참여함으로써
교통약자에 대한 인식 개선 및 지역사회 공감대 확산

## 🎯 Target Users

핵심: 휠체어, 전동 스쿠터 사용자 등 교통약자

확장: 유모차 사용자, 노약자 등 이동에 제약이 있는 모든 시민

참여: 공익 제보를 하고 싶은 일반 시민 및 자원봉사자

## 🆚 Key Differences
### 💡 1) 실시간 동적 장애물 반영

기존 지도: 엘리베이터·경사로 같은 정적 시설만 제공
Bus-ing: 고장, 공사, 불법주차 등 예측 불가 장애물 실시간 반영

### 💡 2) Door-to-Door 통합 이동 지원

기존 앱(위버스 등): 저상버스 탑승 모니터링 정도
Bus-ing: 집 출발 → 목적지 도착까지 전체 이동 과정 지원

### 💡 3) 탑승 과정 장벽 해소

기존: 운전자에게 의사 전달이 어려움
Bus-ing: 앱 기반 탑승 예약 + 자원봉사자 호출로 장벽 최소화

| 기존 서비스 (Limit) | Bus-ing의 차별화된 솔루션 (Solution) |
| :--- | :--- |
| **1. 일반 지도 앱**<br>(네이버 지도, 카카오맵)<br><br> **기본 편의시설 정보에 그침**<br>엘리베이터나 경사로가 '있다'는 정보만 제공할 뿐,<br>현재 고장 났거나 공사 중인 **돌발 상황**은 알 수 없음. | **실시간 '돌발 장애물' 반영**<br>시민들의 실시간 제보를 통해<br>갑작스러운 **공사, 승강기 고장, 불법 주정차** 등<br>예측 불가능한 위험 요소를 즉시 지도에 표시. |
| **2. 위버스 (WeBus)**<br>(저상버스 특화 앱)<br><br> **'정류장 ↔ 정류장'에 국한**<br>버스 탑승 예약 기능에만 집중되어 있어,<br>집에서 정류장까지 가는 길이나<br>하차 후 목적지까지의 이동은 지원하지 못함. | **Door-to-Door (전 과정) 통합 지원**<br>집 문을 나서는 순간부터 목적지 도착까지<br>끊김 없는 경로 안내와 **자원봉사자 매칭**을 통해<br>이동의 모든 과정을 빈틈없이 케어. |
| **3. 서울동행맵**<br>(서울시 공식 앱)<br><br> **행정 데이터의 시차(Time-lag) 발생**<br>민원 접수 후 처리되는 공공 데이터 방식이라,<br>**'지금 당장'** 발생한 도로 위 문제를<br>실시간으로 반영하기에는 한계가 있음. | **집단지성 기반의 즉각적 업데이트**<br>사용자가 직접 참여하고 검증하는 시스템으로<br>데이터 갱신 속도가 매우 빠르며,<br>단순 정보 제공을 넘어 **서로 돕는 커뮤니티** 형성. |

## 🧩 Main Features
### 1️⃣ 시민참여형 장애물 지도

누구나 장애물을 촬영해 위치 기반으로 제보

제보 신뢰도 평가 시스템

제보자 리워드 제공(포인트 등)

### 2️⃣ 장애물 우회 스마트 경로

실시간 장애물 데이터를 기반으로

교통약자 맞춤형 최소 위험 경로 제공

### 3️⃣ 자원봉사자 매칭

자원봉사자 활동 가능 지역·시간 등록

도움이 필요한 이용자와 실시간 매칭

전화/메시지 기반 소통 지원

### 4️⃣ 교통약자 맞춤 이동 보조 서비스

저상버스 탑승 예약/알림

장애인 콜택시 호출

이동 과정 실시간 안내

## 🎉 Expected Impact
